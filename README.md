# patioAutos
Programa web que gestiona conexiones entre ejecutivos y asistentes de patio para rent a car.

## Comandos iniciales
Para comenzar debes realizar los siguiente:  
* Abre una terminal
  * `cd client`
  * `npm i`
* Abre otra terminal
  * `cd server`
  * `npm i`

**En la terminal del server**
`npm run dev`

**En la terninal del cliente**
`npm run dev`
